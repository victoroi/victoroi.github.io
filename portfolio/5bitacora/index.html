Instalación Tema Hugo

Descarga del tema elegido a través de hugo.com

Zip abierto y carpeta agregada a la carpeta website3 de Hugo

Abrir Terminal y ejecutar hugo server dentro de website3

Abrir Visual Studio y confirmar nombre en configtoml sea el mismo que el de la carpeta descargada (zip)

En visual agregar el texto y las imágenes

Para crear dirección de la página web, realizarlo a través de Github:

  1. Crear carpeta public (Terminal - en website - hugo)
  2. Copiar archivos nueva carpeta public y pegar en carpeta Github (victoroi.github.io)
  3. Ir a Github Desktop (app) y poner Commit to master
  4. En Chrome abrir mi nueva dirección creada
